Need: python3

how to import external modules in gdl:
```GDLite
github repo from user import file
```
how to print:
```GDLite
print("string")
```
```GDLite
print(variable)
```
```GDLite
print(f"string {variable}")
```
how to make variable:
```GDLite
var i = "hello"
```
```GDLite
var i = 1
```
```GDLite
var i = True
```
```GDLite
var i = False
```
how to make func(def in python):
```GDLite
func hello():
	print("Hello World!")

hello()
```
how to run your .gdl files:

compile them first:
```
gdlc yourfile.gdl
```
then run the compiled file(python)
```
python yourfile.py
```
